# ScrollWeaver 广场页重构 - 完整实施方案（最终版）

## 一、项目概述

### 1.1 项目目标
重构广场页为“四象阵”布局，提供清晰的四大功能入口，提升用户体验和视觉一致性。

### 1.2 涉及文件清单

#### HTML文件
- `/frontend/pages/plaza.html` - 广场页（主入口，修改）
- `/frontend/pages/creation.html` - 制卷选择页（新建）
- `/frontend/pages/library.html` - 藏书页（新建）
- `/frontend/pages/explore.html` - 阅卷页（修改）
- `/frontend/pages/gathering.html` - 雅集页（新建）

#### CSS文件
- `/frontend/css/pages/plaza.css` - 广场页样式（修改）
- `/frontend/css/pages/creation.css` - 制卷选择页样式（新建）
- `/frontend/css/pages/library.css` - 藏书页样式（新建）
- `/frontend/css/pages/explore.css` - 阅卷页样式（修改）
- `/frontend/css/pages/gathering.css` - 雅集页样式（新建）

#### JavaScript文件
- `/frontend/js/pages/plaza.js` - 广场页交互（修改）
- `/frontend/js/pages/creation.js` - 制卷选择页交互（新建）

- 使用 `linear-gradient` 创建极淡的云状效果
- 使用 `@keyframes` 实现缓慢的水平移动

#### 2.4.2 CSS实现
```css
body::before {
    content: '';
    position: fixed;
    bottom: 0;
    left: 0;
    width: 100%;
    height: 200px;
    background: linear-gradient(
        to top,
        rgba(107, 68, 35, 0.05) 0%,
        rgba(107, 68, 35, 0.02) 50%,
        transparent 100%
    );
    pointer-events: none;
    z-index: 0;
    animation: cloudFlow 20s ease-in-out infinite;
}

@keyframes cloudFlow {
    0%, 100% {
        transform: translateX(0);
        opacity: 0.5;
    }
    50% {
        transform: translateX(50px);
        opacity: 0.8;
    }
}
```

async function loadRooms() {
    try {

function createRoomCard(room) {

async function joinRoom(roomId) {
    const passwordInput = document.getElementById(`password-${roomId}`);

        const response = await fetch(`/api/rooms/${roomId}/join`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': `Bearer ${localStorage.getItem('token')}`
            },
            body: JSON.stringify({ password })
        });

### 8.1 API接口需求

1. `GET /api/user/scrolls` - 获取用户自己的书卷列表

3. `GET /api/scrolls/system` - 获取系统书卷列表
4. `GET /api/rooms` - 获取房间列表
5. `POST /api/rooms/create` - 创建房间
6. `POST /api/rooms/:id/join` - 加入房间
